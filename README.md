## Usage

**CDN link**:

Add to: `site/config/config.php`

```php
c::set('panel.stylesheet', 'https://unpkg.com/kirby-panel-themes/dist/{{ theme_name_here }}.min.css');
```

**NPM install**

```bash
$ npm install --save-dev kirby-panel-themes
```
